# AppfactoryJS CLI

Command line tool to manage AppfactoryJS framework


### Installing

To install run 

```
npm install "@equippedcoding/appfactory"
```


## Getting Started

Create a new appfactoryJS project

```
appfactory create --dir "directory_name" --title "App Title"
```
